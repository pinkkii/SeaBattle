# SeaBattle
Sea battle, connection with 2 players and more

## Features:
- Bot
- To play together
- To play with random
- Classic sea battle

## Installation:
1. Clone this repository `git clone https://github.com/pinkkii/SeaBattle.git`
2. Go to `back` folder `cd back`
3. Run seabattle:
    - With npm: 
      - `npm install`
      - `nodemon index.js or nodemon`
